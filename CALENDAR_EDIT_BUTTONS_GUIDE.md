# Calendar Edit Buttons - Where to Find Them

## For Admins Only

### Location 1: Calendar Grid (Monthly View)
- **Where:** Inside each colored event box on the calendar dates
- **What to look for:** A small **blue square button with a pencil icon** on the RIGHT side of each event
- **Size:** Small blue button (6x6 pixels)
- **Color:** Blue background with white pencil icon

### Location 2: Upcoming Events Section
- **Where:** Scroll down below the calendar grid
- **Section:** "Upcoming Events" list
- **What to look for:** **"Edit"** and **"Delete"** buttons on the RIGHT side of each event card
- **Style:** Blue "Edit" button and Red "Delete" button with text labels

## Troubleshooting

1. **Not seeing buttons?**
   - Make sure you're logged in as an ADMIN user
   - Check if events exist on the calendar
   - Try viewing the calendar in a different month if events are there

2. **Events not showing?**
   - Go to `/admin/events` to create an event first
   - Make sure the event date is in the current month you're viewing

3. **Clear browser cache:**
   - Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
